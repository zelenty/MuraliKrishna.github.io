
# Murali Krishna — Portfolio (GitHub Pages)

This is a ready-to-publish static portfolio site (dark mode) for Murali Krishna.
It is optimized for GitHub Pages and basic SEO (meta tags + JSON-LD).

## What is included
- index.html
- styles.css
- assets/photo.jpg  (your photo)
- resume.pdf (recommended: add your resume PDF here)

## How to publish (free) on GitHub Pages
1. Create a GitHub account if you don't have one: https://github.com
2. Create a new repository named: `muralikrishna.github.io`
3. Upload all files from this zip (or drag-and-drop in the GitHub web UI)
4. Commit and push (or save) — GitHub Pages will serve at: `https://muralikrishna.github.io/`
5. (Optional) Add resume.pdf and any extra project files to the repo.

## Important SEO steps (quick)
- In your repo, open `index.html` and update the canonical URL if you use a custom domain.
- Go to Google Search Console, add the site, and request indexing (optional).
- Share links to LinkedIn, GitHub, Trailhead pointing to your site.
- Add your GitHub Pages link to your LinkedIn profile.
